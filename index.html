A showcase of my professional passions and past work

Web Mapping

My primary focus is presenting geographic information to desicion-makers. Seeing the right data at the right time allows for smarter land management and conservacion actions.

Database Management

Data quality is only as good as the container it lives in. I focus on smart database design that allows for easy data input, analysis, and visualization.

Web Development

Web browsers are an amazing tool to give people access to tool that can help them do thier job better. I build intuitive user-interfaces that allow access to data management tools and geographic analysis that are customized to the user's needs.

Projects

Biodiversity Monitoring Platform

This project was the focus of my Peace Corps service in Mexico. It is a web app that allows field technicions to enter, edit, analyze, and display biodiversity data.

See the code on Github

Recect Phenology

This web app displays the most recent greenness data collect by the MODIS remote sensing platform. Date is displayed as an NDVI difference calcualtion to show how greenness has changed recently. I initially developed this tool for Saguaro National Park, but it now includes all of the contiguous United States.

See the code on Github

Photo Bounties

This project is a platform that allows users to post bounties for photos that they would like to own. Other users submit photos that fufill the poster's specification to collect the bounty. It's a bit like Uber for photos.

See the code on Github (Private Repo)

Saguaro Detection Algorithm

This project's goal was to map the mature saguaros in Saguaro National Park automatically using aerial photos and Python. This produced the first complete map of sagauro locations and density in the park. The research was published in the Journal of Applied Remote Sensing.

See the code on Github

Citizen Science Erosion App

This project was developed to enable visitors at Navajo National Monument to take pictures of erosion sensitive sites and upload them to a cloud database to create a time series for analysis by park staff. This Android app provides offline maps and convinient photo upload capabilities

See the code on Github
